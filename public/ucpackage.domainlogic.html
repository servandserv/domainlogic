1. История школы

Usecases
  • Заходит на сайт
  • Использует 'машину времени'
  • Просматривает ленту документов
  • Просматривает информацию по конкретному документу
Заходит на сайт

URN: ucpackage:domainlogic:In

Пользователь переходит по ссылке http://www..... и попадает на головную страницу сайта. Головная страница сайта предоставляет интерфейсы для навигации по сайту, поисковую строку, 'машину времени'(интерфейс которой состоит из элемента управления и ленты с коллекциями документов).

Использует 'машину времени'

URN: ucpackage:domainlogic:UseTimeMachine

Находясь на главной странице сайта, Пользователь осуществляет навигацию по коллекциям документов за разные периоды времени. Для того, чтобы это сделать он использует элемент управления 'TimeMachine'.

Пользователь использует ползунок этого элемента управления для того, чтобы выбирать искомый год. Для этого он перетаскивает его вверх или вниз. При перетаскивании вниз, отображаемые года меняются от более поздних к более ранним. При перетаскивании вверх, наоборот. результат изменений отображается на цифровом табло 'TimeMachine'. Система обеспечивает переход между годами таким образом, что отображаются только года за которые есть документы а базе данных.

Для перехода на выбранный год Пользователь нажимает кнопку со стрелкой или стопку фотографий над цифровым табло. В результате нажатия лента с коллекциями документов 'прокручивается' до выбранного года.

Просматривает ленту документов

URN: ucpackage:domainlogic:UseDocsFeed

Пользователь просматривает ленту с коллекциями документов за выбранный год. На ленте представлены коллекции документов, каждый документ представлен в виде preview картинки.

  • Документы связанные с классами
  • Документы связанные с преподавателями
  • Документы связанные с мероприятиями
  • Неопознанные документы

В случае, если лента не помещается в экран по высоте, то Пользователь прокручивает ее вниз стандартным для броузера способом. При этом элемент управления 'TimeMachine' скрывается за пределы экрана, а шкала времени, расположенная справа от коллекции документов, остается в неизменном положении.

При нажатии на документ Пользователь переходит к ознакомлению с информацией по этому документу.

Просматривает информацию по конкретному документу

URN: ucpackage:domainlogic:LookAtDocument

Пользователь выбирает документ для просмотра и переходит на страницу информации о документе. Страница предоставляет возможность изучить увеличенное изображение документа, его лицевую и оборотную сторону. Кроме этого документ сопровождает текстовая информация.

Находясь на этой странице Пользователь может вернуться обратно в ленту, или воспользоваться переходом на другие документы выбранного года.
